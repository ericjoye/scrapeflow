# ScrapeFlow — Chrome Web Store Listing

## Extension Name

**ScrapeFlow — AI Web Scraper & Data Exporter**

## Short Description (132 chars max)

Point, click, and export any data from any webpage to CSV or JSON. No code required.

## Long Description

ScrapeFlow is a Chrome extension that lets you point, click, and export structured data from any webpage to CSV or JSON — no code, no selectors, no regex required. If you can see it on the page, you can scrape it.

**ONE-CLICK DATA CAPTURE**
Open the ScrapeFlow popup and click "Start Scraping." Hover over any element on the page — ScrapeFlow highlights it with a blue outline. Click to capture. If the element is inside a detected structure (table, list, or card group), ScrapeFlow offers to extract all matching rows at once. Review your data in the live preview table, then export with one click.

**AI-POWERED AUTO-DETECTION (Pro & Team)**
Pro and Team plans include intelligent structure detection that automatically identifies tables, lists, product cards, and other repeating patterns on any page. Instead of clicking element by element, let AI do the heavy lifting — select a pattern once and extract hundreds of rows instantly.

**CSV & JSON EXPORT**
Free tier supports CSV export for spreadsheet workflows. Pro and Team plans unlock JSON export for developer-friendly integration with databases, APIs, and code. Smart preview lets you review column headers, row counts, and sample values before exporting.

**WORKS EVERYWHERE**
E-commerce listings, directories, search results, real estate, job boards, social media feeds — ScrapeFlow runs on any website via a lightweight Manifest V3 content script. No configuration, no site-specific rules.

**100% LOCAL & PRIVATE**
All data processing happens in your browser. Nothing is sent to external servers. Your scraped data goes directly from the page to your download folder — and nowhere else.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `scrapeflow/` folder
5. The ScrapeFlow icon appears in your toolbar

## Requirements

- Chrome 88+ (Manifest V3 support)
- No additional dependencies
